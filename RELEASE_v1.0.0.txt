INSTALLATION
================================================================================

1. Extract the archive:
   tar -xzf team-metrics-v1.0.0.tar.gz
   cd team-metrics-v1.0.0/

   OR

   unzip team-metrics-v1.0.0.zip
   cd team-metrics-v1.0.0/

2. Follow setup instructions in README.md:
   - Install dependencies: pip install -r requirements.txt
   - Configure: cp config/config.example.yaml config/config.yaml
   - Edit config.yaml with your GitHub/Jira credentials
   - Collect data: python collect_data.py
   - Start dashboard: python -m src.dashboard.app

3. Access at: http://localhost:5000

================================================================================
UPGRADE NOTES
================================================================================

If upgrading from a previous version:
  1. Back up your config/config.yaml
  2. Extract the new release
  3. Restore your config.yaml
  4. Run: pip install -r requirements.txt
  5. Run: python collect_data.py
  6. Restart dashboard

The cache format is compatible - no data migration needed.
